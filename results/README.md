# Results Directory

This folder is used to store outputs generated during model training and evaluation.  
All contents are excluded from version control (see `.gitignore`) to keep the repository lightweight and to avoid committing large files.

## Subdirectories

- **figures/**: Contains all figures generated by the notebooks (`*.tiff`, `*.png`, etc.).  
  These figures correspond to the plots in the manuscript.

- **models/**: Stores trained model checkpoints (`*.pth`).  
  The best model from cross‑validation or final training is saved here.

- **logs/**: Training logs (`*.log` or `*.txt`) including loss curves, validation metrics, and hyperparameter details.

## Notes

- If you need to share results (e.g., for a release), you may archive this folder separately.
- To regenerate all results, run the scripts in the `experiments/` and `notebooks/` directories.
